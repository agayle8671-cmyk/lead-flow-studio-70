## Easiest Way to Start the App

### Option 1: Double-Click Method (Recommended)
Simply **double-click** `START_APP.bat` in your project folder. This will:
- Start the dev server
- Automatically open your browser to http://localhost:8080
- Keep the server running in a separate window

### Option 2: Terminal Commands

**Start dev server and open browser automatically:**
```bash
bun run dev:open
```

**Just start the dev server:**
```bash
bun run dev
```

**Open browser (if server is already running):**
```bash
bun run open
```

### Option 3: Manual Method
1. Open terminal in project folder
2. Run: `bun run dev`
3. Open browser to: http://localhost:8080

---

## App URL
Once running, access your app at:
**http://localhost:8080**

---

## Stopping the Server
- Press `Ctrl+C` in the terminal where the server is running
- Or close the PowerShell window
